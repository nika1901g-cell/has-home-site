# 📋 ИНСТРУКЦИЯ: Как загрузить сайт с админкой на Netlify

## Что у тебя будет в итоге
- Сайт HAS HOME как сейчас
- Вход в /admin через логин и пароль
- Редактирование текстов, акций, товаров прямо в браузере
- Добавление/удаление товаров с фото

---

## 📁 СТРУКТУРА ФАЙЛОВ (создай именно так)

```
твоя-папка/
├── index.html          ← главная страница
├── netlify.toml        ← настройки Netlify
├── admin/
│   ├── index.html      ← страница входа в CMS
│   └── config.yml      ← конфигурация CMS
├── _data/
│   └── settings.json   ← тексты сайта
└── _products/
    └── divan-velour.json ← пример товара
```

---

## ШАГ 1 — Загрузи файлы на GitHub (5 минут)

1. Зайди на https://github.com и создай аккаунт (если нет)
2. Нажми зелёную кнопку **"New"** (создать репозиторий)
3. Название: `has-home-site` → нажми **"Create repository"**
4. Нажми **"uploading an existing file"**
5. Перетащи ВСЕ файлы и папки целиком
6. Нажми **"Commit changes"** (зелёная кнопка внизу)

---

## ШАГ 2 — Подключи GitHub к Netlify (3 минуты)

1. Зайди на https://app.netlify.com
2. Нажми **"Add new site"** → **"Import an existing project"**
3. Нажми **"GitHub"**
4. Выбери репозиторий `has-home-site`
5. Настройки оставь как есть → нажми **"Deploy site"**
6. Подожди ~1 минуту — сайт задеплоится

---

## ШАГ 3 — Включи Netlify Identity (вход в админку)

1. В Netlify открой свой сайт
2. Перейди в **Site settings** (вверху)
3. В левом меню найди **"Identity"**
4. Нажми **"Enable Identity"**
5. Прокрути вниз до **"Registration"** → выбери **"Invite only"**
   (чтобы никто чужой не мог зарегистрироваться)
6. Прокрути ещё ниже до **"Git Gateway"** → нажми **"Enable Git Gateway"**

---

## ШАГ 4 — Пригласи менеджера (создай логин)

1. Остаёшься в разделе **Identity**
2. Нажми **"Invite users"**
3. Введи email менеджера (можно свой)
4. Нажми **"Send"**
5. Менеджер получит письмо → перейдёт по ссылке → задаст пароль

✅ Готово! Теперь менеджер может войти на:
👉 https://splendorous-gecko-c32d91.netlify.app/admin

---

## ШАГ 5 — Как менеджер редактирует сайт

### Первый вход:
1. Открывает https://splendorous-gecko-c32d91.netlify.app/admin
2. Нажимает **"Login with Netlify Identity"**
3. Вводит email и пароль

### Редактировать тексты:
1. В меню слева: **"⚙️ Настройки сайта"**
2. Открывает **"Общие настройки"**
3. Меняет заголовки, слоганы, тексты акции
4. Нажимает **"Publish"** (синяя кнопка вверху)
5. Через ~1 минуту изменения появятся на сайте

### Добавить товар:
1. В меню: **"🛋️ Товары"**
2. Нажимает **"New Products"**
3. Заполняет: название, цена, категория, фото
4. Нажимает **"Publish"**
5. Товар появится на сайте через ~1 минуту

### Удалить товар:
1. В меню: **"🛋️ Товары"**
2. Открывает нужный товар
3. Нажимает **"⋮"** (три точки) → **"Delete"**

## ❓ ЧАСТЫЕ ВОПРОСЫ

**Q: Менеджер забыл пароль?**
A: В Netlify → Identity → найди пользователя → "Reset password"

**Q: Хочу добавить второго менеджера?**
A: Identity → "Invite users" → введи новый email

**Q: Где хранятся данные?**
A: Прямо в GitHub репозитории — надёжно и бесплатно

**Q: Это бесплатно?**
A: Да! Netlify бесплатен для небольших сайтов (до 100GB трафика/месяц)

---

## 🆘 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

- Нет кнопки "Enable Git Gateway" → убедись что включен Identity
- Менеджер не может войти → проверь что он перешёл по ссылке из письма
- Изменения не появляются → подожди 2-3 минуты, Netlify перестраивает сайт
- Ошибка в /admin → проверь что файл config.yml загружен в папку admin/
